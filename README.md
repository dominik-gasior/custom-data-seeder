# custom-data-seeder

The library is used to test C# application. Primary resolves business cases. Helping to tests case which we need real data from business client. Also we can use to other cases if we need it.

## Primary features:
- [ ] fetch data from json and return generate objects
- [ ] mix fake data with real data
- [ ] fetch data from .csv files and generate objects
- [ ] can seed local/test database

The library uses [Bogus library](https://github.com/bchavez/Bogus)
